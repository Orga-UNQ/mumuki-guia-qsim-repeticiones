Si analizamos el ejercicio anterior vemos que hay una **estructura de general de la iteración** (o repetición)

```
       Inicialización
arriba: Controlar condición de ciclo
        Si no se cumple: <salir>
        Cuerpo del ciclo
        Volver <arriba>
salir: Finalizar programa
```
